# 2D 视觉检测模块 (metro_detection)

本模块负责地铁隧道巡检过程中的 2D 图像病害检测。当前阶段为**前期假数据调试阶段**，用于保障整体流水线话题通信的顺畅通车。

---

##  1. 模块功能 (Function)
* 订阅 Gazebo 仿真环境中的相机图像话题。
* 提取图像特征并利用 YOLOv8 算法进行多类别病害识别（如裂缝、渗漏水等）。
* **当前状态**：已编写完备的测试节点 `fake_detection_node.py`，可持续发布严格符合接口规范的模拟病害数据，供三维同化组进行坐标解算联调。

---

##  2. 输入接口 (Inputs)
本模块在正式通车时将订阅以下传感器话题：
* **话题名称**：`/camera/image_raw`
* **消息类型**：`sensor_msgs/msg/Image`
* **含义**：车载相机采集到的实时 RGB 图像流

---

##  3. 输出接口 (Outputs)
本模块将实时发布检测警报，严格契合队长制定的接口规范：
* **话题名称**：`/detections`
* **消息类型**：`std_msgs/msg/String` (标准 JSON 字符串格式)
* **数据样例如下**：
```json
{
  "type": "crack",
  "confidence": 0.95,
  "bbox": [620, 310, 760, 390]
}
